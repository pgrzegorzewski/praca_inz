Cryptosystem with symmetric keys

In this cryptography system both sender and recipent have the same key. They are obliged to choose key before the transmission of the message.
Sender uses key for encrypting message. Recipent uses the same key for decrypting messages.
This the biggest drawback of the system, key for both sides is exactly th same.
