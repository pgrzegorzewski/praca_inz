Cryptanalysis

Cryptanalysis is a branch of science opposite to the cryptography. Cryptanalysis is the study of analyzing the encrypted messages in order to find the true information. Like in the case of cryptography cryptanalysis is strictly connected with the mathematics.
Despite using mathematical algorithms to break the cipher, cryptanalysis also uses the side-channel-attacks. It means that the cryptanalysis finds the weeknesses of the encryption algorithms and exploits vices in the implementation.

There exist four main situations – what kind of information a cryptanalyst can have:
  • Cryptanalyst owns only encrypted text. The most difficult, inconvenient situation for the cryptanalyst
  • Cryptanalyst owns part of encrypted text with coresponding decrypted text
  • Cryptanalyst has temporary access to the encrypting machine
  • Cryptanalyst has temporary access to the decrypting machine

The way of attack on the encrypted text changes respectively to the situation however the aim is always the same – find the key which was used to encrypt the message and then use it to break the cryptogram.
There are three main resources which we have in mind when we attack cryptosystem:
  1. Time – which is required to perform all computation steps, tests to find the key. The less time we need to find the key the more effecting is our attack
  2. Memory – computer's memory required to perform all the computations and cut their time. The more memory we have the less time we spend on breaking the key
  3. Data – amount of intercepted data, not only encrypted text but also fragments of key or original message. The more data we have the easier become the attack to perform

Wiretap types

There are two vulnerabilities concerning exchanging of messages:
  • Active wiretap
  • Passive wiretap

Active wiretap is the vulnerability against the authenticity of the message. The attacker impersonates the sender of the message. The action which prevents spoofing is using the electronical signature.

Passive wiretap is the vulnerability against the confidentiality of the message. The attacker tries to steal the private message. The action which prevents that is using the cipher.
